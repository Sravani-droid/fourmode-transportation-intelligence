# FourMode Transportation Intelligence Platform

Live Demo:
https://fourmode-intelligence.netlify.app/

## Overview

FourMode is an executive transportation intelligence dashboard that consolidates analytics across four major transportation sectors:

- Road Transportation
- Air Transportation
- Rail Transportation
- Maritime Transportation

The platform provides KPI monitoring, comparative analysis, operational insights, and executive-level decision support through an interactive dashboard experience.

---

## Features

### Executive Dashboard
- Transportation Network Health Score
- Cross-mode performance comparison
- Executive insights and recommendations
- KPI summary cards

### Road Analytics
- Revenue analysis
- Route profitability
- Fleet utilization insights
- Carrier performance metrics

### Air Analytics
- Passenger volume analytics
- Cargo movement analysis
- Airline performance indicators
- Regional demand trends

### Rail Analytics
- Freight forecasting
- Capacity planning
- Ton-mile analysis
- Growth intelligence

### Maritime Analytics
- Trade throughput analytics
- Cargo mix analysis
- Port intelligence
- Shipping performance monitoring
